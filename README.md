#CNN - BiLSTM for Arrythmia Detection in ECG Signals
  Trained on MIT BIH Arrythmia Database

#Setup
1. Clone the repo
2. npm install
3. Set up your own X and y data
4. Test model using model.predict(X_test, y_test)
5. Push to GitHub
